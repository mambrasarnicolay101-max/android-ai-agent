## 🚀 Fitur Unggulan Generasi V30.1

### 1. 🔒 Keamanan Zero-Trust Bearer Token
Semua endpoint REST API privat di VPS Alibaba port 80 diproteksi secara ketat menggunakan otentikasi **Bearer Token** (`NOIR_API_KEY`). Dashboard web menyertakan modal glassmorphism **`🔒 LINK NEURAL TERKUNCI`** untuk verifikasi manual, mencegah pemindaian luar tanpa kredensial sah.

### 2. ⚔️ Live Cyber Warfare Arena
Visualisasi radar pertempuran cyberpunk dinamis berbasis HTML5 Canvas 10-vs-10 (Red vs Blue Nodes) dengan sapuan sinar radar, partikel laser, efek monitor CRT retro, serta log mitigasi pertahanan real-time oleh modul Auto-Healer.

### 3. 📱 Onboarding Ramah Pengguna (🌟 Co-Pilot Guide)
Panduan onboarding 3 langkah awal dan kamus perintah operasional lengkap langsung di antarmuka utama, memudahkan adaptasi bagi pengguna baru untuk menyambungkan Android Node, PC Launcher, dan Aegis Guardian.

### 4. 💻 PC Launcher Portabel (`Noir Sovereign.exe`)
Aplikasi desktop portabel Windows (`11.5 MB`) yang dibuat dengan Tkinter + PyInstaller. Aplikasi ini otomatis membaca `.env` lokal, mengecek latensi VPS, dan meluncurkan browser Edge/Chrome dalam **App Mode** dengan menyuntikkan token otentikasi query param secara mulus.

## 🛠️ Panduan Memulai Cepat (Quick Start)

### 1. Hot-Deployment ke VPS Alibaba
Untuk memindahkan pembaruan kode terbaru dan meluncurkan server web FastAPI port 80 di VPS Alibaba secara otonom:
```bash
python deploy_to_vps.py
```

### 2. Menjalankan PC Command Center (Windows)
Double-klik berkas standalone **`Noir Sovereign.exe`** di direktori utama Anda, atau jalankan kode sumbernya secara langsung melalui terminal:
```bash
python pc_launcher.py
```

### 3. Pembersihan Cache & Konflik Repositori
Untuk membersihkan sisa-sisa kompilasi, cache bytecode (`__pycache__`), dan mereset berkas status telemetry yang usang agar workspace tetap rapi dan berperforma tinggi:
```bash
python purge_system.py
```
