    🌟 SELAMAT DATANG DI PUSAT OPERASI NOIR SOVEREIGN

    Noir Sovereign adalah Ekosistem AI Agent Otonom yang memadukan kekuatan komputasi awan (VPS Alibaba) dengan PC ASUS lokal dan perangkat Android native (Aegis) Anda. Halaman panduan ini dirancang untuk memandu pengguna baru agar dapat mengoperasikan sistem dengan sangat mudah tanpa memerlukan latar belakang teknis yang mendalam!

    🚀 3 LANGKAH MUDAH MEMULAI
    1

    Aktifkan Desktop Launcher & Bypass Firewall

    Double-click file Noir Sovereign.exe di Desktop Anda. Aplikasi desktop premium kami otomatis memantau konektivitas VPS Anda di background, melewati popup konsol CMD, dan menampilkan antarmuka ini secara rahasia dan aman.

    2

    Nyalakan PC Override Permission

    Untuk mengizinkan AI Agent Anda menulis kode, menginstal library baru, atau menjalankan kompilasi program secara mandiri di PC Anda, pastikan tombol Toggle PC Override di tab Overview dalam kondisi AKTIF (ON).

    3

    Mulai Membangun Aplikasi Otonom

    Ketikkan keinginan Anda di Link Neural Chat Console menggunakan prefiks perintah /build. Contoh: /build web dashboard toko online minimalis. Duduk santai dan saksikan Agen Anda menulis kode serta mengompilasinya untuk Anda secara langsung!

    💡 TIPS OPERASIONAL CEPAT
    • Ingin Screenshot Ponsel? Cukup ketik perintah /screenshot di konsol obrolan untuk menangkap tampilan ponsel Redmi Note 14 Anda secara instan.
    • Melihat Hasil Build? Buka folder lokal PC di knowledge/sovereign_builds/ untuk melihat kode sumber yang ditulis otonom oleh AI Agent Anda.
    • Evolusi Sistem: Setiap kali AI Agent menyintesis skill baru, proposal evolusi akan muncul di tab 🧬 EVOLUTION. Anda dapat menolak atau menyetujuinya dengan sekali klik!
    📖 KAMUS DIREKTORI PERINTAH UTAMA
    Format Perintah Tujuan Pengoperasian Contoh Penggunaan
    /build [web|exe|android] [spek] Memerintahkan AI Agent merancang & menyusun sistem web, desktop, atau mobile native secara otonom di PC Anda. /build exe aplikasi pemantau baterai
    /override [on|off] Mengizinkan/mencabut hak otonom penuh AI Agent untuk menjalankan perintah lokal dan menginstal pustaka di PC. /override on
    /screenshot Mengambil gambar layar ponsel target (Redmi Note 14) secara langsung dan mengirimkannya ke pusat kendali. /screenshot
    /sysinfo Melakukan audit hardware secara komprehensif untuk menampilkan kapasitas RAM, CPU, dan suhu PC & HP. /sysinfo
    /clean Menginstruksikan modul Auto-Healer untuk membersihkan file cache, logs tak berguna, dan mengoptimalkan memori sistem. /clean
